# 🛒 E-commerce Sales Analysis

## 📌 Overview
This project analyzes an e-commerce dataset to understand sales, profit, and discount trends.

---

## ⚙️ Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

---

## 📊 What I Did
- Cleaned and explored dataset
- Analyzed sales and profit by category
- Calculated profit ratio
- Studied impact of discount on profit
- Identified loss-making products and states

---

## 🔥 Key Insights
- High discounts lead to losses
- Some categories generate high sales but low profit
- Discount strategy directly impacts business profit

---

## 📈 Visualizations
- Scatter plot (Discount vs Profit)
- Bar charts for states and categories

---

## 🎯 Conclusion
This project shows how data analysis can help businesses make better decisions.
